Carousel component.

Example

<ons-carousel style="width: 100%; height: 200px">
    <ons-carousel-item>
     ...
    </ons-carousel-item>
    <ons-carousel-item>
     ...
    </ons-carousel-item>
  </ons-carousel>

Attributes

direction The direction of the carousel. Can be either "horizontal" or "vertical". Default is "horizontal".
fullscreen If this attribute is set the carousel will cover the whole screen.
var Variable name to refer this carousel.
overscrollable If this attribute is set the carousel will be scrollable over the edge. It will bounce back when released.
item-width ons-carousel-item's width. Only works when the direction is set to "horizontal".
item-height ons-carousel-item's height. Only works when the direction is set to "vertical".
auto-scroll If this attribute is set the carousel will be automatically scrolled to the closest item border when released.
swipable If this attribute is set the carousel can be scrolled by drag or swipe.
disabled If this attribute is set the carousel is disabled.
initial-index Specify the index of the ons-carousel-item to show initially. Default is 0.

Methods

next() Show next ons-carousel item.
prev() Show previous ons-carousel item.
first() Show first ons-carousel item.
last() Show last ons-carousel item.
setSwipable(swipable) Set whether the carousel is swipable or not.
isSwipable() Returns whether the carousel is swipable or not.
setActiveCarouselItemIndex(index) Specify the index of the ons-carousel-item to show.
getActiveCarouselItemIndex() Returns the index of the currently visible ons-carousel-item.
setAutoScrollEnabled(enabled) Enable or disable "auto-scroll" attribute.
isAutoScrollEnabled() Returns whether the "auto-scroll" attribute is set or not.
setOverscrollable(overscrollable) Set whether the carousel is overscrollabe or not.
isOverscrollable() Returns whether the carousel is overscrollabe or not.
on(eventName,listener) Add an event listener. Preset events are "postchange" and "refresh".
refresh() Update the layout of the carousel. Used when adding ons-carousel-items dynamically to the carousel or to automatically adjust the size.
isDisabled() Returns whether the dialog is disabled or enabled.
setDisabled(disabled) Disable or enable the dialog.

ons-list-item

Component that represents each item in the list. Must be put inside the ons-list component.

  • Usage
  • Live Example

  • Preset Modifiers

Usage # 

<ons-list>
  <ons-list-header>Header Text</ons-list-header>
  <ons-list-item>Item</ons-list-item>
  <ons-list-item>Item</ons-list-item>
</ons-list>

Attributes #

Name Type
Default Value		 Description
modifier String The appearance of the list item. Optional
lock-on-drag String Prevent vertical scrolling when the user drags horizontally. Optional

Preset Modifiers #

Modifier Description
tight Remove the space above and below the item content. This is useful for multi-line content.
tappable Make the list item change appearance when it's tapped.
chevron Display a chevron at the right end of the list item and make it change appearance when tapped.

ons-modal

Modal component that masks current screen. Underlying components are not subject to any events while the modal component is shown.

  • Live Example

Attributes #

Name Type
Default Value		 Description
var String Variable name to refer this modal. Optional
animation String
default 		The animation used when showing and hiding the modal. Can be either "none" or "fade". Optional
animation-options Expression Specify the animation's duration, timing and delay with an object literal. E.g. {duration: 0.2, delay: 1, timing: 'ease-in'} Optional

Methods Summary #

Signature Description
toggle([options]) Toggle modal visibility.
show([options]) Show modal.
hide([options]) Hide modal.
getDeviceBackButtonHandler() Retrieve the back button handler.

ons.platform

現在実行されているプラットフォームを検知するためのユーティリティメソッドを収めたオブジェクトです。

  • メソッド
  • 議論

メソッド概要 #

シグネチャ 概要
isWebView() Cordova内で実行されているかどうかを返します。
isIOS() iOS上で実行されているかどうかを返します。
isAndroid() Android上で実行されているかどうかを返します。
isIPhone() iPhone上で実行されているかどうかを返します。
isIPad() iPad上で実行されているかどうかを返します。
isBlackBerry() BlackBerry上で実行されているかどうかを返します。
isOpera() Opera上で実行されているかどうかを返します。
isFirefox() Firefox上で実行されているかどうかを返します。
isSafari() Safari上で実行されているかどうかを返します。
isChrome() Chrome上で実行されているかどうかを返します。
isIE() Internet Explorer上で実行されているかどうかを返します。
isIOS7above() iOS7以上で実行されているかどうかを返します。

ons-list-item

リストの各要素を表現するためのコンポーネントです。ons-listコンポーネントと共に使用します。

  • 使い方
  • 実例
  • 属性
  • プリセットのmodifier

使い方 # 

<ons-list>
  <ons-list-header>Header Text</ons-list-header>
  <ons-list-item>Item</ons-list-item>
  <ons-list-item>Item</ons-list-item>
</ons-list>

属性 #

名前 型 / デフォルト値 概要
modifier String 各要素の表現を指定します。 Optional

プリセットのmodifier #

Modifier名 概要
tight 行間のスペースを取り除きます。複数行の内容をリストで扱う場合に便利です。
tappable タップやクリックした時に効果が表示されるようになります。
chevron 要素の右側に右矢印が表示されます。また、タップやクリックした時に効果が表示されるようになります。

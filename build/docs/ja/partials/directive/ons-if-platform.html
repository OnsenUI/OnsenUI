ons-if-platform

プラットフォームやブラウザーに応じてコンテンツの制御をおこないます。ios, android, blackberry, chrome, safari, firefox, operaのいずれかの値を空白区切りで複数指定できます。

  • 使い方
  • 属性
  • 関連情報
  • 議論

使い方 # 

<div ons-if-platform="android">
  ...
</div>

関連情報 #

  • 他のユーティリティAPI
  • ons-if-orientationコンポーネント

属性 #

名前 型 / デフォルト値 概要
ons-if-platform String "opera", "firefox", "safari", "chrome", "ie", "android", "blackberry", "ios", "wp"のいずれか空白区切りで複数指定できます。 Optional

ons-list

使い方 # 

<ons-list>
  <ons-list-header>Header Text</ons-list-header>
  <ons-list-item>Item</ons-list-item>
  <ons-list-item>Item</ons-list-item>
</ons-list>

属性 #

名前 型 / デフォルト値 概要
modifier String リストの表現を指定します。 Optional

プリセットのmodifier #

Modifier名 概要
inset 親要素の画面いっぱいに広がらないリストを表示します。
noborder リストの上下のボーダーが無いリストを表示します。

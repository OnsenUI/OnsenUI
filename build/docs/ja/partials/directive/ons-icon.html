ons-icon

アイコンを表示するコンポーネントです。Font AwesomeもしくはIoniconsから選択できます。

  • 使い方
  • 実例
  • 属性
  • 関連情報

使い方 # 

<ons-icon
  icon="fa-twitter"
  size="20px"
  fixed-width="false"
  style="color: red">
</ons-icon>

属性 #

名前 型 / デフォルト値 概要
icon String アイコン名を指定します。fa-で始まるものはFont Awesomeとして、ion-で始まるものはIoniconsとして扱われます。使用できるアイコンはこちら: http://fontawesome.io/icons/　および　http://ionicons.com。 Optional
size String アイコンのサイズを指定します。値は、lg, 2x, 3x, 4x, 5xもしくはピクセル単位で指定できます。 Optional
rotate Number アイコンを回転して表示します。90, 180, 270から指定できます。 Optional
flip String アイコンを反転します。horizontalもしくはverticalを指定できます。 Optional
fixed-width Boolean
false 		等幅にするかどうかを指定します。trueもしくはfalseを指定できます。デフォルトはfalseです。 Optional
spin Boolean
false 		アイコンを回転するかどうかを指定します。trueもしくはfalseを指定できます。 Optional

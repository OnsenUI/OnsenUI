ons-page

ページ定義のためのコンポーネントです。このコンポーネントの内容はスクロールが許可されます。

  • 使い方
  • 属性
  • メソッド
  • 関連情報
  • 議論

使い方 # 

<ons-page>
  <ons-toolbar>
    <div class="center">Title</div>
  </ons-toolbar>

  ...
</ons-page>

関連情報 #

  • 複数のページを管理する
  • ページ初期化のイベント
  • バックボタンに対応する
  • CSSスタイルのオーバーライド
  • 複数のページを1つのHTMLに記述する

属性 #

名前 型 / デフォルト値 概要
var String このページを参照するための名前を指定します。 Optional
modifier String スタイル定義をカスタマイズするための名前を指定します。 Optional
on-device-backbutton Expression デバイスのバックボタンが押された時の挙動を設定できます。 Optional
ng-device-backbutton Expression デバイスのバックボタンが押された時の挙動を設定できます。AngularJSのexpressionを指定できます。 Optional

メソッド概要 #

シグネチャ 概要
getDeviceBackButtonHandler() バックボタンハンドラを取得します。このメソッドはnullを返す場合があります。

メソッド

getDeviceBackButtonHandler(): Object #

バックボタンハンドラを取得します。このメソッドはnullを返す場合があります。

返り値: デバイスのバックボタンハンドラを返します。

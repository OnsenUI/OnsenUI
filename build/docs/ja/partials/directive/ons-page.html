ons-page

ページ定義のためのコンポーネントです。このコンポーネントの内容はスクロールが許可されます。

  • 使い方
  • 属性
  • メソッド
  • イベント
  • 関連情報
  • 議論

使い方 # 

<ons-page>
  <ons-toolbar>
    <div class="center">Title</div>
  </ons-toolbar>

  ...
</ons-page>

関連情報 #

  • 複数のページを管理する
  • ページライフサイクルイベント
  • バックボタンに対応する
  • CSSスタイルのオーバーライド
  • 複数のページを1つのHTMLに記述する

属性 #

名前 型 / デフォルト値 概要
var String このページを参照するための名前を指定します。 Optional 初期化時のみ有効 angular
modifier String スタイル定義をカスタマイズするための名前を指定します。 Optional
on-device-backbutton Expression デバイスのバックボタンが押された時の挙動を設定できます。 Optional angular
ng-device-backbutton Expression デバイスのバックボタンが押された時の挙動を設定できます。AngularJSのexpressionを指定できます。 Optional 初期化時のみ有効 angular
ons-init Expression "init"イベントが発火された時の挙動を独自に指定できます。 Optional 初期化時のみ有効 angular
ons-show Expression "show"イベントが発火された時の挙動を独自に指定できます。 Optional 初期化時のみ有効 angular
ons-hide Expression "hide"イベントが発火された時の挙動を独自に指定できます。 Optional 初期化時のみ有効 angular
ons-destroy Expression "destroy"イベントが発火された時の挙動を独自に指定できます。 Optional 初期化時のみ有効 angular

メソッド概要 #

シグネチャ 概要
getDeviceBackButtonHandler() バックボタンハンドラを取得します。このメソッドはnullを返す場合があります。

イベント概要 #

名前 概要
init ページがアタッチされた後に発火します。
show ページが表示された後に発火します。
hide ページが隠れた後に発火します。
destroy ページが破棄される前に発火します。

メソッド

getDeviceBackButtonHandler(): Object #

バックボタンハンドラを取得します。このメソッドはnullを返す場合があります。

返り値: デバイスのバックボタンハンドラを返します。

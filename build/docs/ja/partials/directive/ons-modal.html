ons-modal

画面全体をマスクするモーダル用コンポーネントです。下側にあるコンポーネントは、 モーダルが表示されている間はイベント通知が行われません。

使い方 # 

<ons-modal>
  ...
</ons-modal>

属性 #

名前 型 / デフォルト値 概要
var String このモーダルを参照するための名前を指定します。 Optional
animation String
default 		モーダルを表示する際のアニメーション名を指定します。"none"もしくは"fade"を指定できます。 Optional
animation-options Expression アニメーション時のduration, timing, delayをオブジェクトリテラルで指定します。e.g. {duration: 0.2, delay: 1, timing: 'ease-in'} Optional

メソッド概要 #

シグネチャ 概要
toggle([options]) モーダルの表示を切り替えます。
show([options]) モーダルを表示します。
hide([options]) モーダルを非表示にします。
isShown() モーダルが表示されているかどうかを返します。
getDeviceBackButtonHandler() ons-modalに紐付いているバックボタンハンドラを取得します。

メソッド

toggle([options]) #

モーダルの表示を切り替えます。

パラメーター
名前 型 概要
options Object オプションを指定するオブジェクト。
options.animation String アニメーション名を指定します。"none", "fade"のいずれかを指定します。
options.animationOptions String アニメーション時のduration, delay, timingを指定します。e.g. {duration: 0.2, delay: 0.4, timing: 'ease-in'}

show([options]) #

モーダルを表示します。

パラメーター
名前 型 概要
options Object オプションを指定するオブジェクト。
options.animation String アニメーション名を指定します。"none", "fade"のいずれかを指定します。
options.animationOptions String アニメーション時のduration, delay, timingを指定します。e.g. {duration: 0.2, delay: 0.4, timing: 'ease-in'}

hide([options]) #

モーダルを非表示にします。

パラメーター
名前 型 概要
options Object オプションを指定するオブジェクト。
options.animation String アニメーション名を指定します。"none", "fade"のいずれかを指定します。
options.animationOptions String アニメーション時のduration, delay, timingを指定します。e.g. {duration: 0.2, delay: 0.4, timing: 'ease-in'}

isShown(): Boolean #

モーダルが表示されているかどうかを返します。

返り値: モーダルが表示されている場合にtrueとなります。

getDeviceBackButtonHandler(): Object #

ons-modalに紐付いているバックボタンハンドラを取得します。

返り値: デバイスのバックボタンハンドラを返します。

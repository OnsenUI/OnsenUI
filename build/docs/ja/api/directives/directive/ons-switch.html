スイッチを表示するコンポーネントです。

属性

disabled スイッチを無効の状態にする場合に指定します。
checked スイッチがONの状態にするときに指定します。
var JavaScriptから参照するための変数名を指定します。
modifier カスタムなスタイルを適用するための名前を指定します。

メソッド

isChecked() スイッチがONの場合にtrueを返します。
setChecked(isChecked) スイッチの値を指定します。isCheckedにはtrueもしくはfalseを指定します。
getCheckboxElement() スイッチが内包する、input[type=checkbox]の要素を取得します。
on(eventName,listener) イベントリスナーを追加します。changeイベントを使用できます。

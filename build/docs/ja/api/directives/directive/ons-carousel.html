カルーセルを表示できるコンポーネント。

例

<ons-carousel style="width: 100%; height: 200px">
    <ons-carousel-item>
     ...
    </ons-carousel-item>
    <ons-carousel-item>
     ...
    </ons-carousel-item>
  </ons-carousel>

属性

direction カルーセルの方向を指定します。horizontalかverticalを指定できます。デフォルトはhorizontalです。
fullscreen この属性があると、absoluteポジションを使ってカルーセルが自動的に画面いっぱいに広がります。
var このカルーセルを参照するための変数名を指定します。
overscrollable この属性がある時、タッチやドラッグで端までスクロールした時に、バウンドするような効果が当たります。
item-width ons-carousel-itemの幅を指定します。direction属性にhorizontalを指定した時のみ有効です。
item-height ons-carousel-itemの高さを指定します。direction属性にverticalを指定した時のみ有効です。
auto-scroll この属性がある時、一番近いcarosel-itemの境界まで自動的にスクロールするようになります。
swipable この属性がある時、カルーセルをスワイプやドラッグで移動できるようになります。
disabled この属性がある時、dargやtouchやswipeで受け付けなくなります。
initial-index 最初に表示するons-carousel-itemを0始まりのインデックスで指定します。デフォルトは0です。

メソッド

next() 次のons-carousel-itemを表示する。
prev() 前のons-carousel-itemを表示する。
first() 最初のons-carousel-itemを表示する。
last() 最後のons-caroulse-itemを表示する。
setSwipable(swipable) swipable属性があるかどうかを設定する。
isSwipable() swiapble属性があるかどうかを返す。
setActiveCarouselItemIndex(index) 表示するons-carousel-itemをindexで指定する。
getActiveCarouselItemIndex() 現在表示されているons-caroulse-itemのindexを返す。
setAutoScrollEnabled(enabled) auto-scroll属性があるかどうかを設定する。
isAutoScrollEnabled() auto-scroll属性があるかどうかを返す。
setOverscrollable(overscrollable) overscroll属性があるかどうかを設定する。
isOverscrollable() overscroll属性があるかどうかを返す。
on(eventName,listener) イベントリスナーを追加します。"postchange"か"refresh"を指定できます。
refresh() レイアウトや内部の状態を最新のものに更新する。ons-caroulse-itemを動的に増やしたり、ons-carouselの大きさを動的に変える際に利用する。
isDisabled() disabled属性があるかどうかを返す。
setDisabled(disabled) disabled属性があるかどうかを設定する。

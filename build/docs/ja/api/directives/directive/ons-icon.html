アイコンを表示するコンポーネントです。Font Awesomeから選択できます。

属性

icon アイコン名を指定します。先頭の`fa-`の部分は取り除いてください。使用できるアイコンはこちら: http://fontawesome.io/icons/。
size アイコンのサイズを指定します。値は、lg, 2x, 3x, 4x, 5xもしくはピクセル単位で指定できます。
rotate アイコンを回転して表示します。90, 180, 270から指定できます。
flip アイコンを反転します。horizontalもしくはverticalを指定できます。
fixed-width リスト内で使う場合に、trueを指定すると縦に整列します。trueもしくはfalseを指定できます。デフォルトはtrueです。
spin アイコンを回転するかどうかを指定します。trueもしくはfalseを指定できます。

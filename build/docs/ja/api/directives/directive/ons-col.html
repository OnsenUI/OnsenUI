グリッドシステムにて列を定義します。ons-rowとともに使用し、コンポーネントの配置に使用します。

属性

align 縦の配置を指定します。次の値から選択してください: top, center, bottom
width カラムの横幅を指定します。パーセントもしくはピクセルで指定します（10%や50px）。

注意: Android 4.3以前、もしくはiOS 6以前のOSの場合、ons-rowとons-columnを組み合わせた場合に描画が崩れる場合があります。
